FAQ

Di mana data saya disimpan?

Semua entri Anda disimpan secara lokal di perangkat Anda menggunakan SQLite bawaan iOS. Tidak ada yang dikirim ke server kami karena kami tidak memilikinya.

Bisakah saya mencadangkan atau memindahkan entri saya ke perangkat lain?

Sinkronisasi iCloud tidak tersedia di v1.0. Data Anda hanya dipulihkan jika Anda mencadangkan perangkat itu sendiri melalui iCloud Backup atau iTunes/Finder. Kami mungkin menambahkan fitur ekspor eksplisit di versi mendatang.

Apa yang dibuka oleh pembelian Pendukung?

Pada v1.0, pembelian Pendukung memberikan lencana kecil di profil Anda dan membantu mendanai pengembangan. Pembelian ini tidak membuka fitur apa pun di v1.0. Jika versi mendatang menambahkan fitur khusus Pendukung, para pendukung yang ada akan menerimanya secara otomatis.

Bagaimana cara memulihkan pembelian saya di perangkat baru?

Buka Aplikasi, masuk ke Pengaturan, dan ketuk Pulihkan Pembelian. Pastikan Anda masuk dengan Apple ID yang sama dengan yang Anda gunakan untuk pembelian awal.

Bagaimana cara meminta pengembalian dana?

Apple menangani semua pengembalian dana untuk pembelian App Store. Kunjungi https://reportaproblem.apple.com.

Bagaimana cara menghapus semua data saya?

Hapus instalasi Aplikasi dari perangkat Anda. iOS akan menghapus semua data yang disimpan secara lokal secara otomatis.
